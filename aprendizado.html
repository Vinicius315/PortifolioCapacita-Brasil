Dicas Gerais para Desenvolvedores

  • Usar nomenclaturas como padrão para nomear arquivos e propriedades, como kebab-case e camelCase.
  • Sempre se manter atualizado com mudanças nas documentações das tecnologias.
  • Escreva código limpo, legível e bem descritivo.
  • Utilize controle de versão com Git.
  • Gerencie bem o tempo, usando técnicas como o Pomodoro.
  • Desenvolva projetos pessoais regularmente.
  • Estude tecnologias atuais e demandadas no mercado.

Trechos de Código

Comandos Git Básicos

git init
git add .
git status
git commit -m "Primeiro commit"
git log
git clone [URL]
git config --global user.email
git config --global user.name

Criar uma Ramificação

git checkout -b nome-da-branch
git branch
git merge nome-da-branch
git push origin nome-da-branch

Enviar Projeto para o GitHub

git remote add origin [URL]
git push -u origin main
